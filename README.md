# Dino Rush

Dino Rush is a fun game that can be played in the browser.

## Live Game

Please checkout the live Game here [Dino Rush](https://asif-mahmud25.github.io/dino-rush/)

## Technologies Used

* HTML5
* CSS3
* JavaScript ES6

No framework has been used, only pure HTML, CSS & Javascript.
